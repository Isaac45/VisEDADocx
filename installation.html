Installation

Install the core package for image EDA, or install optional extras for hyperspectral and point cloud workflows.

Core installation

pip install viseda

Optional installs

pip install "viseda[hyperspectral]"
pip install "viseda[pointcloud]"
pip install "viseda[report]"
pip install "viseda[all]"

Development install

git clone https://github.com/your-org/viseda.git
cd viseda
pip install -e ".[all,dev]"
Note: the current pyproject.toml uses placeholder repository URLs. Replace these before publishing.
