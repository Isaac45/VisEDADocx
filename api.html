API Reference

Function-focused reference for the current VisionEDA public API, with ImageEDA documented in the greatest detail because it is the complete part of the library.

ImageEDA public methods

MethodPurposeTypical use
ImageEDA(...)Creates an image EDA object with configuration for colour clustering, duplicate detection, blur detection, texture, and frequency analysis.Instantiate once per dataset or experiment.
load(source, labels=None, label_from_parent=False, recursive=True)Loads image files from a directory, file, or list of paths.Use for normal folder-based datasets.
load_arrays(arrays, labels=None)Loads image arrays directly from memory.Use for generated, augmented, or externally loaded images.
summary()Returns all major EDA statistics as a dictionary.Best for programmatic analysis and automated quality checks.
normalization_stats()Returns RGB mean and std in [0, 1].Use in model preprocessing pipelines.
plot()Creates the full EDA dashboard.Use after the internal radar-axis issue is patched.
plot_colour()Creates a colour-analysis dashboard.Inspect colour distributions and dataset colour bias.
plot_quality()Creates a quality-analysis dashboard.Use after the internal radar-axis issue is patched.
plot_texture()Creates texture feature histograms.Inspect GLCM features for surface/detail patterns.
plot_samples()Displays random sample images.Visually verify dataset content.
plot_class_samples()Displays sample images per class.Compare classes visually.
plot_average_image()Computes average and standard-deviation images.Find spatial bias and object-position consistency.
plot_channel_correlation()Plots RGB channel histograms and pairwise scatter plots.Check channel relationships and colour preprocessing issues.
plot_duplicates(mode="exact"|"near")Visualizes duplicate or near-duplicate image groups.Review possible leakage or repeated images.
report(output_path)Generates an HTML report.Share results outside Python.
