The missing EDA toolkit for non-tabular visual data.

VisEDA provides exploratory data analysis for image datasets, hyperspectral scenes, and 3D point clouds. This documentation package is built around the current codebase, with viseda/image/eda.py and testimg.py treated as the most complete reference implementation.

ImageEDA

Inventory, corrupt-image checks, spatial statistics, pixel distributions, quality metrics, colour analysis, texture, frequency, duplicate detection, dashboards, and reports.

HyperspectralEDA

Cube-level analysis, spectral signatures, PCA, NDVI, false-colour previews, band statistics, and dataset-level cube summaries.

PointCloudEDA

3D geometry, bounding boxes, density, height profiles, intensity/RGB distributions, spatial outliers, and 2D/3D visualisation.
